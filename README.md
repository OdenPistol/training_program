# ⚡ WAR MACHINE — 30-Day Training Program

Programme d'entraînement de 30 jours avec suivi d'exercices, nutrition, suppléments et glossaire.

## Stack
- React 19 + Vite
- Mobile-first, PWA-ready
- localStorage pour la persistence

## Setup

```bash
npm install
npm run dev
```

## Deploy (GitHub Pages)

```bash
npm run build
```

Push le dossier `dist/` sur la branche `gh-pages`, ou configure GitHub Pages pour servir depuis la branche `main` → dossier `/dist`.

## Specs
- **89kg → 82-83kg** en 30 jours
- **~20% → 13-14% BF**
- Split: Pecs/Tri → Cali → Épaules/Bi → Cardio+Sortie → Repos → Jambes → Dos/Pecs
- 3×30min cardio post-training
- Jeûne 36h semaines 1-2
- ~2250 kcal/jour training
